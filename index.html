Frequently Asked Questions

What is a dynamic QR code?

A dynamic QR code is a QR code whose destination URL can be changed after it has been printed. Unlike static QR codes, dynamic QR codes also provide scan analytics including location, device, and time data. Learn more about dynamic QR codes.

Is QRRQ QR code generator free?

Yes, QRRQ offers a free plan that includes 3 dynamic QR codes, unlimited static QR codes, and basic scan analytics. Premium plans are available for additional features like custom designs, password protection, and API access. See pricing plans.

What types of QR codes can I create?

QRRQ supports 15+ QR code types including URL, PDF, vCard, WiFi, Email, Phone, SMS, WhatsApp, Location, Event, Text, Social Media, Multi-URL, App Store, and Crypto. Try it free.

Can I track QR code scans?

Yes, QRRQ provides real-time scan analytics including total scans, geographic location, device type, operating system, browser, and time-based analysis. Data can be exported as CSV.

Can I add a logo to my QR code?

Yes, you can customize your QR code with custom colors, logo upload, different patterns, eye shapes, frames, and call-to-action text. Your QR codes will match your brand identity perfectly.

What is a Link Page?

A Link Page is a micro landing page you can create with QRRQ. It supports 12 block types including links, headers, social media, YouTube, Spotify, contact forms, and maps. Perfect for sharing multiple links through one QR code. Explore Link Pages.

Does QRRQ have an API?

Yes, QRRQ provides a full REST API for QR code creation, management, and analytics. Available on Pro and Business plans. View API documentation.
